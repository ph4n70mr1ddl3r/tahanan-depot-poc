# 07 — ROLES AND JOB DESCRIPTIONS

## 7.1 Role Coding Convention

All roles follow the pattern: `TD-[ENTITY]-[DEPARTMENT]-[LEVEL]-[SEQ]`

- Entity: HDL=Holdings, RET=Retail, SCM=Supply Chain, PROP=Property, DIG=Digital
- Level: C=C-Suite, VP=Vice President, D=Director, M=Manager, S=Supervisor, A=Associate
